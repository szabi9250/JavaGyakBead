Létrehozandó adatbázis neve: beadando
beadando.sql file segítségével a mellékelt táblák és adatok beimportálása

Az adatbázishoz beadando.sql által létrehozott táblák:

Süti tábla
CREATE TABLE suti (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nev VARCHAR(255) NOT NULL,
    tipus VARCHAR(100) NOT NULL,
    dijazott BOOLEAN NOT NULL DEFAULT FALSE
);

Tartalom tábla
CREATE TABLE tartalom (
    id INT AUTO_INCREMENT PRIMARY KEY,
    sutiid INT NOT NULL,
    mentes VARCHAR(50),
    FOREIGN KEY (sutiid) REFERENCES suti(id)
        ON DELETE CASCADE
        ON UPDATE CASCADE
);

Ár tábla
CREATE TABLE ar (
    id INT AUTO_INCREMENT PRIMARY KEY,
    sutiid INT NOT NULL,
    ertek INT NOT NULL,
    egyseg VARCHAR(50),
    FOREIGN KEY (sutiid) REFERENCES suti(id)
        ON DELETE CASCADE
        ON UPDATE CASCADE
);
